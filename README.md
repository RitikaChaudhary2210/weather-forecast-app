# Weather Forecast App

Android application that displays a **3-day weather forecast** for a selected city.

## Features

* Search weather by city
* 3-day forecast display
* Weather icons and temperature
* Offline support using Room database
* Jetpack Compose UI

## Tech Stack

* Kotlin
* Jetpack Compose
* Retrofit
* Moshi
* Room Database
* MVVM Architecture

## Setup

1. Clone the repository
2. Add OpenWeather API key in `local.properties`

Example:

WEATHER_API_KEY=your_api_key_here

3. Run the app
